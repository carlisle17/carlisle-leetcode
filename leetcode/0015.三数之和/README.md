## 题目： 三数之和

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？

请你找出所有和为 0 且不重复的三元组。 **注意：答案中不可以包含重复的三元组。**

#### 示例 1：

    输入：nums = [-1,0,1,2,-1,-4]
    输出：[[-1,-1,2],[-1,0,1]]
    
#### 示例 2：

    输入：nums = []
    输出：[]
    
#### 示例 3：

    输入：nums = [0]
    输出：[]
    
#### 提示：

    0 <= nums.length <= 3000
    -105 <= nums[i] <= 105
    
## 解题分析：

- 暴力解法：三层嵌套循环，时间复杂度为O(n^3)。
- 排序+双指针：当一个数确定时，就变成了两数之和。与“两数之和”不同之处在于，该题是求解数值而非索引，故可以打乱顺序，即可以使用排序。
